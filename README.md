# 🎓 Campus Recruitment Web Application

A full-stack web application designed to simplify and automate the campus recruitment process. This platform enables students to apply for jobs, recruiters to manage job postings, and administrators to oversee the entire hiring workflow.

## 🚀 Features

### 👨‍🎓 Student
- Register and login securely
- View available job opportunities
- Apply for jobs
- Track application status

### 🏢 Recruiter
- Post job openings
- View student applications
- Shortlist candidates

### 🛠️ Admin
- Manage users (students & recruiters)
- Monitor recruitment activities
- Control platform access

## 🧰 Tech Stack

- Frontend: React.js
- Backend: Spring Boot
- Database: MySQL
- API: REST APIs
